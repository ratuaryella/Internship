## Configuration
File directory:

```diff
    -- api
    -- config
    -- -- config.js
    -- app.js
    -- package.json
    -- README.md
    -- sequelize.js
    -- server.js
```

Example configuration. Add file '.js' with name: **config.js** to folder config.
```diff
module.exports = {
	
}
```

### Get Package
```diff
    npm install
```

### Running
```diff
    npm run start
```
