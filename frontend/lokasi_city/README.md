## Configuration
File directory:

```diff
    -- controllers
    -- fetch
    -- middleware
    -- public
    -- router
    -- validator
    -- views
    -- app.js
    -- package.json
    -- README.md
    -- sequelize.js
```

Example configuration. Add file '.js' with name: **config.js** to folder config.
```diff
module.exports = {

}
```

### Get Package
```diff
    npm install
```

### Running
```diff
    npm run start
```
